# Quagmire

> **The internet is a web. Follow the threads.**

Quagmire is a full-stack AI-powered knowledge graph explorer that maps how internet culture, ideas, trends, and rabbit holes connect. Type any topic and watch it unfold into an interactive web of concepts, communities, creators, events, and ideologies — then keep clicking deeper.



---

## What It Does

Most tools give you a list of results. Quagmire gives you a map.

Type "dopamine detox" and you don't just get a definition — you get a visual graph showing how it connects to Andrew Huberman, productivity culture, TikTok motivation discourse, the manosphere, overstimulation anxiety, and a dozen other threads you didn't know were related. Click any node to go deeper. Follow the threads as far as you want.

- **Map connections** — see how ideas link across internet culture
- **Follow rabbit holes** — click any node to expand it into new territory
- **Discover surprises** — AI surfaces unexpected connections marked with ✦

## How the AI Works

Every graph is generated by Gemini using a carefully engineered prompt system:

1. **Generate** — user types a topic, Gemini returns 8-12 nodes with categories, descriptions, seeds, obscurity levels, and edges
2. **Transform** — raw JSON is validated and converted to ReactFlow nodes/edges with dagre auto-layout
3. **Expand** — clicking a node sends it back to Gemini with the full canvas context, returns 2-3 new connected nodes
4. **Images** — after each generation, Wikipedia is queried in parallel for thumbnail images

Each node has:
- **Category** — concept, community, creator, trend, platform, event, opposition, aesthetic, ideology, media
- **Obscurity level** — 1 (obvious) → 2 (interesting) → 3 (surprising, marked with ✦)
- **Seeds** — 3 rabbit hole suggestions for further exploration
- **Description** — written in Quagmire's voice: internet-native, curious, never academic

## Security

- API key is server-side only — never exposed to the client
- Input sanitization blocks prompt injection patterns before they reach Gemini
- Rate limiting: 10 generate calls / 30 expand calls per IP per minute
- `server-only` enforced on AI modules to prevent accidental client bundling
- Payload size limits on `existingLabels` to prevent prompt bloat
